# coffee_shop_sales_dashboard
## Overview
This Excel dashboard analyzes coffee shop sales, customer footfall, and product performance to generate business insights.
## Tools Used
- Microsoft Excel
- Pivot Table & Charts
## Dataset
Includes transaction data, product categories, total bill, etc
## Key Insights
- Peak sales occure in morning hours
- Coffee category contributes highest sales
- Barista Espresso is the top-selling product
- Hell's Kitchen has highest footfall
- Weekday sales are higher than weekends
## Dashboard Features 
- Quantity Sold by Hour
- Sales by Category
- Footfall by Store
- Size Distribution
- Top 5 Products
- Orders by Weekday
